To the users of this website:

This website is a one-person project created to produce the first comprehensive catalog of NIFB (New Independent Fundamental Baptist) websites. This project operates without external support; because of this, the extent of all NIFB resources remains unknown. What distinguishes this project is its focus on being a directory rather than a hosting platform (I make exceptions with certain documents). While I currently lack the budget to host sermons and videos, I will continue to provide valuable links to websites offering these resources. I sincerely appreciate any help you can provide in expanding this project. If you find anything yet to be cataloged or encounter broken links (link rot), please don't hesitate to contact me at Directory.NIFB@aol.com. I extend my gratitude to everyone who has contributed to this project thus far.
